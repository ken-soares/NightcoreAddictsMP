Introduction

NightcoreAddictsMP is an mp3 player that supports playlist making, OBS, XSplit and i3blocks integration (as well as any other software that can read a text file) but it is also quite customizable. Below, you'll find some user-made themes.



"Default" by HaiiroKoneko & Eliral


Here is a copy of the default theme if you ever mess something up in your config file and want to revert.

Default config.json file:

{ "default-color": "#000000",
"warning-color": "#FF0000",
"__comment": "!!max font size is 14 for regular-font-size",
"title-font-size": 16,
"regular-font-size": 11,
"bg-color": "#DBD9DB",
"font": "Verdana",
"symbols": true,
"startup_sound": true,
"bg-button-color": "#DBD9DB",
"__comment3": "!!File path must include / if on Windows",
"default-file-path": "",
"default-volume": 0.5,
"playlist-dir": "playlists/",
"logo-image": "Misc/logo_small.png"
}
